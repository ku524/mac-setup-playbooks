# mac-setup-playbooks

## How-to-use
### 1. fill out variable file - `inventory/group_vars/all.yml`
```yaml
git_email:
git_userName: 
```

### 2. Login AppStore

### 3. Run the [install shell script](https://github.com/tensult/mac-setup-playbooks/blob/master/install.sh) in the terminal to run the ansible code.
```sh
$ ./install.sh
```
